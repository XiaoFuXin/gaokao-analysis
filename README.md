# 高考录取数据分析 —— 院校专业组投档趋势与择校策略

## 项目简介

本项目以江西省 2021-2023 年本科一批普通类录取数据为基础，结合软科 2026 中国大学排名，从“录取位次波动风险”、“院校性价比”和“专业热度变化”三个维度进行量化分析，为考生志愿填报提供数据支持。

## 技术栈

- Python 3
- MySQL
- Requests（爬虫，调用公开 API）
- PyMySQL（数据库操作）
- Pandas / NumPy（数据处理）
- Matplotlib（可视化）

## 项目结构

├── crawler.py # 爬虫脚本：批量获取全国高校在江西的本科一批专业录取数据

## 数据来源

- **录取数据**：来自中国教育在线官方公开 API（`static-data.gaokao.cn`），均为本科一批普通类录取信息，经过严格清洗，保留物理类、历史类共 14,121 条有效记录，覆盖 985/211 高校在江西的招生专业。
- **排名数据**：来自软科官方 API（`shanghairanking.cn`）2026 年中国大学排名主榜，取前 500 所。

## 分析维度

### 1. 录取位次波动风险（变异系数）

计算每所学校和每个专业近三年录取位次的**变异系数**（标准差/均值），量化报考不确定性。

- 输出：
  - 专业波动最大 Top50（高风险专业）
  - 专业波动最小 Top50（稳定专业）
  - 院校波动最大 Top20
  - 院校波动最小 Top20

### 2. 院校性价比指数

将软科排名与平均录取位次归一化后求差值，构建**性价比指数**，寻找“实力强但录取门槛相对较低”的院校。

- 输出：
  - 性价比最高的 15 所院校
  - 性价比最低的 15 所院校

### 3. 专业热度变化趋势

对比 2021 年与 2023 年各专业录取位次中位数的相对变化，计算**热度指数**。

- 输出：
  - 热度上升最快的 10 个专业（越来越难考）
  - 热度下降最快的 10 个专业（越来越容易考）

## 使用方式

1. 安装依赖：`pip install -r requirements.txt`
2. 配置数据库：修改 `config.py` 中的 MySQL 连接信息，创建数据库 `gaokao_db`
3. 运行爬虫：`python crawler.py`（首次运行建议只爬一年数据测试）
4. 清洗数据：`python data_cleaner.py`
5. 爬取软科排名：`python crawl_ranking.py`
6. 执行分析：`python analysis.py`，图表将保存至 `outputs/figures/`

## 分析结论（部分摘要）

- **高波动专业**：主要集中在顶尖医学院（临床医学、口腔医学）和外语类院校，因其极低的录取位次基数而呈现出高位波动，报考时需关注最新排名。
- **稳定选择**：中分段工科（土木、环境、机械）和师范类专业位次波动极低，是求稳考生的理想选择。
- **性价比之选**：多所中部地区“双一流”及行业特色院校凭借较强的学科实力和相对温和的录取位次，成为“价值洼地”。
- **热度转向**：人工智能、大数据等新工科专业位次近年显著上升，而传统外语、土木类专业热度下降，反映考生就业预期的变化。
